# cloud-mall-cloud Agent Instructions

规则文件按主题拆分：

| 文件 | 内容 |
|------|------|
| `.agents/build.md` | 构建命令、测试、本地开发、模块结构 |
| `.agents/code-style.md` | 命名、import、格式、集合、MP 查询 |
| `.agents/domain-convention.md` | 包结构、分层、异常、数据源、API 设计 |
| `.agents/checklist.md` | 提交前检查清单 |
